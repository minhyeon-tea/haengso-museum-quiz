# 행소박물관 유물 4지선다 퀴즈

첨부 문서의 프로젝트 기획을 바탕으로 만든 `pygame` 퀴즈입니다. 문서에는 실제 문항 목록이 없어 행소박물관의 역사, 전시실, 주요 유물을 주제로 15문항을 구성했습니다.

퀴즈 문항, 화면 구성, 입력 처리 등 모든 파이썬 코드는 `main.py` 하나에 들어 있습니다.

## 실행 방법

- main.py 실행

## 조작 방법

- 마우스 클릭: 보기 선택
- `Esc`: 종료

## 포함 기능

- 행소박물관 관련 4지선다 15문항
- 정답·오답 색상 및 텍스트 피드백
- 진행률과 실시간 점수
- 최종 결과 메시지
- Windows 한국어 폰트 자동 선택
